# Lab3 - Personal Expenses Tracker

This project is a **Personal Expenses Tracker** created in Python.  
It allows users to:

- Track their balance
- Add new expenses
- View expenses by item name or amount
- Optionally archive old expenses

## Files Included

- `Expenses_tracker.py` - Main Python script for tracking expenses.
- `archive_expenses.sh` - Shell script to archive expense files (optional).

## How to Run

1. Ensure Python 3 is installed.
2. Open terminal in the project folder.
3. Run the script:

```bash
python Expenses_tracker.py
